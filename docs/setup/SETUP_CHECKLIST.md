# Unity Collective - Setup Checklist

Use this checklist to ensure a smooth setup and launch of your Unity Collective website.

## üéØ Phase 1: Initial Setup (30 minutes)

### Prerequisites Installation
- [ ] **Node.js 18+** installed ([Download](https://nodejs.org/))
- [ ] **pnpm** installed (`npm install -g pnpm`)
- [ ] **Git** installed ([Download](https://git-scm.com/))
- [ ] **Code editor** ready (VS Code recommended)

### Project Setup
- [ ] **Download/clone** the Unity Collective project
- [ ] **Navigate** to project folder (`cd unity-collective`)
- [ ] **Install dependencies** (`pnpm install`)
- [ ] **Start dev server** (`pnpm run dev`)
- [ ] **Verify** website loads at `http://localhost:5173`

### Initial Testing
- [ ] **Homepage** displays correctly
- [ ] **Navigation** works between pages
- [ ] **Business Directory** page loads
- [ ] **Community Hub** page loads
- [ ] **Mobile view** looks good (use browser dev tools)
- [ ] **All images** loading properly

## üé® Phase 2: Customization (1-2 hours)

### Content Updates
- [ ] **Update business listings** in `src/App.jsx` (featuredBusinesses array)
- [ ] **Modify community stats** (communityStats object)
- [ ] **Add your events** (upcomingEvents array)
- [ ] **Update contact information** in footer
- [ ] **Add your social media links**

### Branding Customization
- [ ] **Replace logo** (`src/assets/logo_main.png`)
- [ ] **Update hero banner** (`src/assets/hero_banner.png`)
- [ ] **Customize colors** (if needed) in `src/App.css`
- [ ] **Update page title** in `index.html`
- [ ] **Add favicon** to `public/` folder

### Content Review
- [ ] **Proofread all text** content
- [ ] **Verify business information** is accurate
- [ ] **Check event dates** and details
- [ ] **Confirm contact information**
- [ ] **Test all internal links**

## üöÄ Phase 3: Pre-Launch (1 hour)

### Technical Preparation
- [ ] **Build project** (`pnpm run build`)
- [ ] **Test production build** (`pnpm run preview`)
- [ ] **Check for console errors**
- [ ] **Verify all pages work** in production build
- [ ] **Test on different browsers** (Chrome, Firefox, Safari)

### SEO & Analytics Setup
- [ ] **Google Analytics account** created
- [ ] **Analytics tracking code** added to `index.html`
- [ ] **Meta descriptions** updated
- [ ] **Open Graph tags** added for social sharing
- [ ] **Sitemap** generated (optional)

### Domain & Hosting
- [ ] **Domain name** purchased (if using custom domain)
- [ ] **Hosting platform** account created (Netlify, Vercel, etc.)
- [ ] **DNS settings** ready (if using custom domain)
- [ ] **SSL certificate** plan confirmed

## üåê Phase 4: Deployment (30 minutes)

### Choose Your Deployment Method
- [ ] **Option A: Netlify** (drag & drop `dist` folder)
- [ ] **Option B: Vercel** (CLI deployment)
- [ ] **Option C: GitHub Pages** (Git integration)
- [ ] **Option D: Traditional hosting** (FTP upload)

### Deployment Steps
- [ ] **Deploy to hosting platform**
- [ ] **Verify site loads** at temporary URL
- [ ] **Test all functionality** on live site
- [ ] **Configure custom domain** (if applicable)
- [ ] **Enable HTTPS/SSL**

### Post-Deployment Verification
- [ ] **All pages accessible**
- [ ] **Images loading correctly**
- [ ] **Navigation working**
- [ ] **Mobile responsiveness** confirmed
- [ ] **Analytics tracking** active
