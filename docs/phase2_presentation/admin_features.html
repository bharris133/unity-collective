SIX KEY FEATURES POWER THE ADMIN REVIEW EXPERIENCE

01

PENDING QUEUE

A dedicated tab surfaces all submissions with a 'pending' status, complete with a live badge count in the sidebar to alert admins of new applications.

02

REVIEW DETAIL PANE

A two-pane UI allows admins to select a submission and view the full business profile, self-declared Black-Owned status, and uploaded verification documents.

03

APPROVE ACTION

A single click sets the verification status to 'verified', instantly propagating the green Verified badge to the Business Directory and Member Dashboard.

04

REJECT WITH REASON

A two-step rejection flow prevents accidental clicks and provides an optional textarea to log the specific reason for rejecting an application.

05

OVERRIDE CAPABILITY

Admins retain full control with an "Override — Approve Anyway" function, allowing previously rejected applications to be verified if issues are resolved.

06

REAL-TIME COUNTERS

Summary cards for Pending, Verified, and Rejected counts update instantly as actions are taken, providing a live snapshot of the platform's verification state.
