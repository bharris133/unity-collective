## üõ†Ô∏è **TECHNICAL SETUP DETAILS**

### Prerequisites
- **Node.js** (version 18 or higher)
- **pnpm** (recommended) or npm
- **Git** for version control
- **Code Editor** (VS Code recommended)

### Installation Steps
```bash
# 1. Extract the complete package
tar -xzf unity-collective-complete-package.tar.gz
cd unity-collective

# 2. Install dependencies
pnpm install

# 3. Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# 4. Start development server
pnpm run dev

# 5. Build for production
pnpm run build
```

### Environment Configuration
```bash
# Firebase Configuration
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project_id

# Stripe Configuration
VITE_STRIPE_PUBLISHABLE_KEY=your_stripe_key

# Email Service
VITE_EMAIL_SERVICE_ID=your_email_service
```

### Deployment Options
1. **Netlify**: Drag and drop `dist` folder
2. **Vercel**: Connect Git repository
3. **AWS S3**: Upload build files
4. **Traditional Hosting**: FTP upload
